# 🌐 Настройка прокси для смены IP

## Зачем нужно

Avito блокирует IP при большом количестве запросов. Чтобы избежать бана нужно:
- Использовать прокси для смены IP
- Парсить небольшими сессиями
- Делать перерывы между сессиями

## Вариант 1: Использование прокси

### Настройка прокси в `parser/.env`:

```env
PORT=3300
MAX_PAGES=3
SIMULATION_SLOWDOWN=2.5

# Прокси настройки
PROXY_SERVER=http://your-proxy-ip:port
PROXY_USERNAME=your_username
PROXY_PASSWORD=your_password
```

### Изменение в коде parser/index.js:

Найдите строку с `args:` и добавьте:

```javascript
args: [
  // Если используете прокси
  ...(process.env.PROXY_SERVER ? [`--proxy-server=${process.env.PROXY_SERVER}`] : []),
  
  '--disable-blink-features=AutomationControlled',
  // ... остальные args
]
```

Если прокси требует авторизацию, добавьте после `await page.setUserAgent(...)`:

```javascript
if (process.env.PROXY_USERNAME && process.env.PROXY_PASSWORD) {
  await page.authenticate({
    username: process.env.PROXY_USERNAME,
    password: process.env.PROXY_PASSWORD
  });
}
```

## Вариант 2: Мобильный интернет / USB модем

Если у вас есть мобильный интернет:
1. Подключите телефон через USB (режим модема)
2. Используйте этот интернет для парсинга
3. При бане - переключите режим "В самолете" на 30 сек (смена IP)

## Вариант 3: Ротация через несколько прокси

Создайте файл `parser/proxies.json`:

```json
[
  "http://proxy1.example.com:8080",
  "http://proxy2.example.com:8080",
  "http://proxy3.example.com:8080"
]
```

## 🛡️ Стратегия безопасного парсинга

### Рекомендуемая схема:

**Сессия 1: утро (9:00-10:00)**
- Парсить 200 объявлений
- Настройки: BATCH_SIZE=2, BATCH_DELAY_MS=5000
- Время: ~40 минут

**Перерыв: 2-3 часа**

**Сессия 2: день (13:00-14:00)**
- Парсить следующие 200 объявлений
- Смените IP (VPN/прокси/перезагрузите роутер)
- Очистите cookies: CLEAR_BROWSER_DATA=1

**Перерыв: 2-3 часа**

**Сессия 3: вечер (17:00-18:00)**
- Парсить оставшиеся
- Снова смените IP

### Ультра-безопасный режим:

```env
# detail-parser/.env
BATCH_SIZE=1                # По 1 объявлению
BATCH_DELAY_MS=8000         # 8 секунд между запросами

# parser/.env
MAX_PAGES=1
SIMULATION_SLOWDOWN=3.0
CLEAR_BROWSER_DATA=1        # Очищать cookies при каждом запуске
```

## 🚨 Что делать если забанили ПРЯМО СЕЙЧАС

1. **Остановите парсер** (Ctrl+C)

2. **Смените IP:**
   - Включите VPN
   - Или перезагрузите роутер
   - Или используйте мобильный интернет

3. **Очистите cookies:**
   ```bash
   # Windows
   rmdir /s /q parser\browser-data
   
   # Или в parser/.env установите:
   CLEAR_BROWSER_DATA=1
   ```

4. **Увеличьте задержки:**
   ```env
   BATCH_SIZE=1
   BATCH_DELAY_MS=10000
   SIMULATION_SLOWDOWN=4.0
   ```

5. **Подождите 30-60 минут**

6. **Парсите малыми партиями**: по 50-100 объявлений за раз

## 📊 Ограничения Avito (примерные)

- **~100-200 запросов** за короткое время = мягкое предупреждение
- **~300-500 запросов** подряд = временный бан на 30-60 минут
- **~1000+ запросов** подряд = бан на несколько часов/дней
