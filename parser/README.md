# Парсер веб-страниц с Puppeteer

Проект для парсинга веб-страниц с максимальным обходом детекции ботов и сохранением авторизации между сеансами.

## Особенности

- ✅ **Обход детекции ботов** с помощью puppeteer-extra-plugin-stealth
- ✅ **Сохранение авторизации** между запросами (cookies, localStorage)
- ✅ **Имитация реального пользователя** (движения мышью, прокрутка, задержки)
- ✅ **Видимый браузер** для ручной авторизации
- ✅ **HTTP API** для удобного использования
- ✅ **CORS поддержка** для интеграции с фронтендом

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Сервер запустится на `http://localhost:3000`

## Использование

### Базовый запрос

```bash
curl "http://localhost:3000?url=https://example.com"
```

### Из браузера

```
http://localhost:3000?url=https://example.com
```

### Из JavaScript

```javascript
const response = await fetch('http://localhost:3000?url=https://example.com');
const html = await response.text();
console.log(html);
```

## Авторизация на сайтах

1. Запустите сервер: `npm start`
2. Сделайте первый запрос к нужному сайту
3. В открывшемся браузере вручную авторизуйтесь
4. Авторизация автоматически сохранится в папке `browser-data`
5. Все последующие запросы будут использовать сохраненную авторизацию

## Технические детали

### Структура проекта

```
avito_demo_pu/
├── index.js           # Основной файл сервера
├── package.json       # Зависимости и настройки
├── .gitignore         # Игнорируемые файлы
├── browser-data/      # Данные браузера (автоматически создается)
└── README.md          # Документация
```

### Настройки браузера

- Используется режим НЕ headless (видимый браузер)
- UserDataDir для сохранения сессий
- Отключены флаги автоматизации
- Реалистичный User-Agent и заголовки
- Viewport 1920x1080

### Имитация пользователя

- Случайные задержки (3000-4500 мс)
- Случайные движения мышью
- Прокрутка страницы
- Переопределение navigator.languages

## Переменные окружения

Создайте файл `.env` для настройки:

```env
PORT=3000
```

## Остановка сервера

Нажмите `Ctrl+C` для корректного завершения работы и закрытия браузера.
